===============================================
   BillMaster Pro - Portable Billing Software
===============================================

QUICK START:
1. Double-click "Start-BillMaster.bat"
2. Wait for your web browser to open automatically
3. Application runs at http://localhost:8080
4. Keep the command window open while using the app

FILES INCLUDED:
- BillMaster.exe (Server application)
- spa/ folder (Web interface - REQUIRED!)
- Start-BillMaster.bat (Launcher)
- README.txt (This file)

FEATURES:
✓ Bill generation with zero mismatches
✓ Auto-calculated bill numbers
✓ Customer transaction import (Excel)
✓ Stock inventory management
✓ Multi-format export (PDF, Excel, HTML)
✓ Account separation for multiple businesses
✓ HTML report customization
✓ Completely offline operation
✓ All data stored locally

REQUIREMENTS:
- Windows 10/11
- Web browser (Chrome, Firefox, Edge recommended)
- 150MB disk space
- No internet connection required

USAGE NOTES:
- All data is stored in your browser's local storage
- Each account/business has separate data
- Close the black command window to stop the server
- The spa/ folder must stay next to BillMaster.exe

TROUBLESHOOTING:
- If browser doesn't open: Manually go to http://localhost:8080
- If "port in use" error: Close the app and try again
- If files missing: Ensure spa/ folder is present
